# Events
This document defines the trigger conditions and resolution of events.

---

## ARTICLE 1 – EVENTS

### §1.1 – Triggers
Events are triggered by:
- Passage of time (e.g., seasonal events);
- Domain thresholds being reached;
- Card effects;
- Player actions.

### §1.2 – Structure
Each event shall specify:
- **Trigger**: the condition that causes the event to occur;
- **Effect**: the changes to Domains, Population, resources, or Figureheads;
- **Warning flag**: if the event may affect Figureheads, the text shall so indicate without giving odds.

### §1.3 – Resolution
Events resolve immediately when triggered, within the current Tick.
